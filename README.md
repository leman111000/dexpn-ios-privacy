# DexPN iOS — Legal Pages (Privacy / Support / Deletion)

This repository hosts the public legal pages for the DexPN iOS app, published via GitHub Pages:

- Privacy Policy: `/privacy-policy`
- Privacy Choices: `/privacy-choices`
- Account Deletion: `/account-deletion`
- Support: `/support`

## Setup (GitHub Pages)
1. Repo → Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, folder: / (root)

After publishing, use the generated URLs in App Store Connect:
- Privacy Policy URL → `https://<<github_user>>.github.io/<<repo_name>>/privacy-policy`
- User Privacy Choices URL → `https://<<github_user>>.github.io/<<repo_name>>/privacy-choices`
- Support URL (if needed) → `https://<<github_user>>.github.io/<<repo_name>>/support`

## Replace placeholders
Search for `<<...>>` and replace:
- `<<support_email>>`
- `<<support_telegram>>` (optional)
- `<<company_name>>`
- `<<website_url>>` (optional)
